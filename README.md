# s21_decimal

Реализация типа `decimal` на языке C с точной арифметикой для финансовых и высокоточных вычислений.

## 📌 О проекте

`s21_decimal` — это библиотека, реализующая собственный тип данных `decimal`, аналогичный decimal из других языков (например, C#).

Основная цель — избежать ошибок округления, характерных для `float` и `double`, что особенно важно в финансовых расчетах.

Тип представлен структурой:

```c
typedef struct {
    int bits[4];
} s21_decimal;
```

* 96 бит — мантисса
* 1 бит — знак
* 8 бит — степень (scale)

## ⚙️ Возможности

### ➕ Арифметические операции

* `s21_add` — сложение
* `s21_sub` — вычитание
* `s21_mul` — умножение
* `s21_div` — деление

---

### 🔍 Операторы сравнения

* `s21_is_less`
* `s21_is_less_or_equal`
* `s21_is_greater`
* `s21_is_greater_or_equal`
* `s21_is_equal`
* `s21_is_not_equal`

---

### 🔄 Преобразования

* `s21_from_int_to_decimal`
* `s21_from_float_to_decimal`
* `s21_from_decimal_to_int`
* `s21_from_decimal_to_float`

---

### 🧮 Дополнительные функции

* `s21_floor` — округление вниз
* `s21_round` — округление до ближайшего
* `s21_truncate` — отбрасывание дробной части
* `s21_negate` — смена знака

---

## 🧠 Особенности реализации

* Побитовая работа с мантиссой
* Поддержка scale
* Банковское округление (round half to even)
* Обработка переполнений
* Использование расширенного типа (`big_decimal`) для вычислений

## 🚀 Сборка и запуск

Клонировать репозиторий:

```bash
git clone https://github.com/SteveBarnesss/s21_decimal.git
cd s21_decimal
```

Сборка библиотеки:

```bash
make
```

Запуск тестов:

```bash
make test
```

Проверка покрытия:

```bash
make gcov_report
```
